# Getting Started

First, run the development server:

```bash
cp .env .env_example
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


For tests:

```bash
npm run test
```

For lint:

```bash
npm run lint
```

For build:

```bash
npm run build
```
